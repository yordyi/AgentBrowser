## Overview

AgentBrowser is positioned as **"The Local Browser Runtime for AI Agents"** with key differentiators:

- **Ref System** - 93%+ token savings vs screenshot-based approaches
- **Local-First** - Privacy-focused, no cloud dependency required
- **iOS Native** - Only browser automation tool with Mobile Safari support
- **MCP Integration** - Native support for Claude Code and AI agent protocols

## Business Model

Recommended path: **Open Source + Hosted Service + Enterprise License**

```
Phase 1: Developer Ecosystem (0-6 months)
Phase 2: Hosted Service (6-12 months)
Phase 3: Enterprise Version (12-24 months)
```
